# Facebook Auto Account Creator Tool

## Documentation
This tool automates the creation of Facebook accounts, simplifying the process for users looking to set up multiple accounts quickly.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ayeshakhan7647/Ayeshafbcreator.git
   ```
2. **Install dependencies:**
   Navigate into the cloned directory and run:
   ```bash
   npm install
   ```
3. **Set up configuration:**
   Edit the `config.json` file with necessary credentials and settings.

## Usage Guide
1. Run the tool from your terminal:
   ```bash
   node index.js
   ```
2. Follow the prompts to enter necessary information.

## Features
- Automates the account creation process.
- Allows bulk account creation.
- User-friendly interface with clear instructions.
- Handles common errors during account creation.
